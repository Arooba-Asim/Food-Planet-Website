Food Planet Website

Food Planet is a responsive front-end food ordering website developed using HTML, CSS, and Bootstrap. The website is designed to provide users with an engaging and user-friendly platform for exploring food items, viewing menus, and enjoying a seamless online food ordering experience.

The project focuses on modern web design principles, responsive layouts, and intuitive navigation to ensure a smooth browsing experience across desktops, tablets, and mobile devices.

Features
- Responsive design for all screen sizes
- Attractive homepage and food menu sections
- User-friendly navigation
- Modern and clean interface
- Organized layout for easy browsing
- Mobile-friendly design

Technologies Used
- HTML
- CSS

Project Objective
The objective of this project was to design and develop a visually appealing food ordering website that enhances user experience through responsive design, easy navigation, and an attractive presentation of food items.
